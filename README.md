Bike-3D-Game
============

A Bike Game in 3D with mountain terrain and fossils to collect 

A terrain With A Bike Which Can be moved in all the directions .
Controls :
1 : Camera view 1
2 : Camera View 2
3 : Camera View 3
4 : Camera View 4
5 : Camera View 5
Arrow Keys : Movement

To Run :
Type Make On the terminal
Type ./project
